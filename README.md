## 🚀 Déploiement

### Prérequis
```bash
node >= 18
npm ou pnpm
```

### Installation locale
```bash
git clone https://github.com/Med12-q/HUB-BUG-SYSTEM.git
cd HUB-BUG-SYSTEM
npm install
npm run dev
```

### Build production
```bash
npm run build
# Output → dist/
```

### Déploiement Vercel
```bash
# Via Vercel CLI
npm i -g vercel
vercel --prod

# Ou connecter le repo directement sur vercel.com
```
